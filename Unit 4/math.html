Q1. Prove that: |1 a a²| = (a-b)(b-c)(c-a)

Given Determinant:

|1 a a²|

|1 b b²|

|1 c c²|

Solution:

Step 1: Apply R₂ → R₂ - R₁ and R₃ → R₃ - R₁

|1 a a² |

|0 b-a b²-a² |

|0 c-a c²-a² |

Step 2: Factor out common terms

Note: b² - a² = (b-a)(b+a) and c² - a² = (c-a)(c+a)

|1 a a² |

|0 b-a (b-a)(b+a)|

|0 c-a (c-a)(c+a)|

Step 3: Take (b-a) from R₂ and (c-a) from R₃

= (b-a)(c-a) × Determinant

|1 a a² |

|0 1 b+a |

|0 1 c+a |

Step 4: Expand along column 1

= (b-a)(c-a) × 1 × |1 b+a|

= (b-a)(c-a) × [(c+a) - (b+a)]

= (b-a)(c-a) × (c-b)

= (b-a)(c-a)(c-b)

= (a-b)(b-c)(c-a) [Rearranging signs]

Hence Proved: |1 a a²| = (a-b)(b-c)(c-a)

Q2. Find the value of x for which the matrix is singular

Matrix: |1 1 x |

|-2 2 2 |

|3 1 -3 |

Solution:

A matrix is singular when its determinant = 0

Step 1: Calculate the determinant

|A| = 1|2 2 | - 1|-2 2 | + x|-2 2|

|1 -3 | |3 -3| |3 1|

Step 2: Expand each 2×2 determinant

= 1[(2)(-3) - (2)(1)] - 1[(-2)(-3) - (2)(3)] + x[(-2)(1) - (2)(3)]

= 1[-6 - 2] - 1[6 - 6] + x[-2 - 6]

= 1[-8] - 1[0] + x[-8]

= -8 + 0 - 8x

= -8 - 8x

Step 3: Set determinant = 0

-8 - 8x = 0

-8x = 8

x = -1

Answer: x = -1

Q7. Prove that the matrix is orthogonal

Matrix: A = (1/9)|−8 1 4|

|4 4 7|

|1 −8 4|

Solution:

A matrix A is orthogonal if AAᵀ = I (or AᵀA = I)

Step 1: Find Aᵀ

Aᵀ = (1/9)|−8 4 1|

|1 4 −8|

|4 7 4|

Step 2: Calculate AAᵀ

AAᵀ = (1/81)|−8 1 4||−8 4 1|

|4 4 7||1 4 −8|

|1 −8 4||4 7 4|

Step 3: Compute each element

(1,1): (64 + 1 + 16)/81 = 81/81 = 1

(1,2): (-32 + 4 + 28)/81 = 0/81 = 0

(1,3): (-8 - 8 + 16)/81 = 0/81 = 0

(2,1): (-32 + 4 + 28)/81 = 0/81 = 0

(2,2): (16 + 16 + 49)/81 = 81/81 = 1

(2,3): (4 - 32 + 28)/81 = 0/81 = 0

(3,1): (-8 - 8 + 16)/81 = 0/81 = 0

(3,2): (4 - 32 + 28)/81 = 0/81 = 0

(3,3): (1 + 64 + 16)/81 = 81/81 = 1

AAᵀ = |1 0 0| = I

|0 1 0|

|0 0 1|

Hence Proved: The matrix is orthogonal

Q8. Find x, y, z if the matrix is orthogonal

Matrix: A = |0 x x |

|2y y -y |

|z -z z |

Solution:

For orthogonal matrix: Each row (and column) must be a unit vector, and rows must be orthogonal

Condition 1: Each row has magnitude 1

Row 1: √(0² + x² + x²) = 1

√(2x²) = 1

2x² = 1

x² = 1/2

x = ±1/√2

Row 2: √(4y² + y² + y²) = 1

√(6y²) = 1

6y² = 1

y² = 1/6

y = ±1/√6

Row 3: √(z² + z² + z²) = 1

√(3z²) = 1

3z² = 1

z² = 1/3

z = ±1/√3

Condition 2: Rows must be orthogonal (dot product = 0)

Row 1 · Row 2 = 0(2y) + x(y) + x(-y) = 0 ✓

Row 1 · Row 3 = 0(z) + x(-z) + x(z) = 0 ✓

Row 2 · Row 3 = 2y(z) + y(-z) + (-y)(z) = 2yz - yz - yz = 0 ✓

Answer:

x = ±1/√2, y = ±1/√6, z = ±1/√3

Signs can be chosen independently as long as orthogonality is maintained
